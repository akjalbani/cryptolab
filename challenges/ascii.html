ASCII Conversion

Learn to convert integers to characters using the ASCII standard - the foundation of text encoding.

Background

ASCII (American Standard Code for Information Interchange) is a 7-bit encoding standard that represents text using integers 0-127. Each character maps to a unique number, making it the foundation of modern text encoding.

Character Range Decimal Hex
A-Z (uppercase) 65-90 0x41-0x5A
a-z (lowercase) 97-122 0x61-0x7A
0-9 (digits) 48-57 0x30-0x39
{ (open brace) 123 0x7B
} (close brace) 125 0x7D

Challenge

Using the integer array below, convert each number to its corresponding ASCII character to reveal the flag.

Integer Array
[99, 114, 121, 112, 116, 111, 123, 65, 83, 67, 73, 73, 95, 112, 114, 49, 110, 116, 52, 98, 108, 51, 125]
üí°
Hint
In Python, chr(n) converts an integer to its ASCII character, and ord(c) does the reverse.

Solution

Python Code

Python 
ords = [99, 114, 121, 112, 116, 111, 123, 65, 83, 67, 
        73, 73, 95, 112, 114, 49, 110, 116, 52, 98, 
        108, 51, 125]

flag = ''.join(chr(o) for o in ords)
print(flag)
