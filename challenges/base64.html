CHALLENGE 03 EASY

Base64 Encoding

Transform binary data into a portable ASCII format using Base64.

Background

Base64 encodes binary data using 64 printable ASCII characters (A-Z, a-z, 0-9, +, /). Each Base64 character represents 6 bits, so 4 Base64 characters encode 3 bytes (24 bits).

The term "ASCII armor" refers to encoding binary encrypted data in Base64 so it can be safely transmitted through text-only channels like email.

Challenge

Take this hex string, decode it into bytes, then encode it into Base64.

Hex String
72bca9b68fc16ac7beeb8f849dca1d8a783e8acf9679bf9269f7bf

Solution

Python 
import base64

hex_string = '72bca9b68fc16ac7beeb8f849dca1d8a783e8acf9679bf9269f7bf'

# Step 1: Hex to bytes
raw_bytes = bytes.fromhex(hex_string)

# Step 2: Bytes to Base64
result = base64.b64encode(raw_bytes).decode()
print(result)

Conversion Pipeline:

Hex String ‚Üí bytes.fromhex() ‚Üí Raw Bytes ‚Üí b64encode() ‚Üí Base64 String
