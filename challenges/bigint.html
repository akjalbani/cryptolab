CHALLENGE 04 MEDIUM

Bytes & Big Integers

Convert between messages and large integers - essential for RSA cryptography.

Background

Cryptosystems like RSA work on numbers, but messages are text. The standard approach is to convert each character to its ASCII byte, combine into hexadecimal, then interpret as a decimal integer.

Example: Converting "HELLO"

Step Value
MessageHELLO
ASCII bytes[72, 69, 76, 76, 79]
Hex bytes[0x48, 0x45, 0x4c, 0x4c, 0x4f]
Combined hex0x48454c4c4f
Decimal310400273487

Challenge

Convert this integer back into a message:

Large Integer
11515195063862318899931685488813747395775516287289682636499965282714637259206269

Hint
PyCryptodome's long_to_bytes() function handles this conversion. Install with pip install pycryptodome

Solution

Method 1: Using PyCryptodome

Python 
from Crypto.Util.number import long_to_bytes

n = 11515195063862318899931685488813747395775516287289682636499965282714637259206269
result = long_to_bytes(n)
print(result)

Method 2: Manual Conversion

Python 
n = 11515195063862318899931685488813747395775516287289682636499965282714637259206269

# Convert to hex, remove '0x' prefix
hex_string = hex(n)[2:]

# Convert hex to bytes
result = bytes.fromhex(hex_string)
print(result)
