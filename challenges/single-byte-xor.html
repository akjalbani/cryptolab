CHALLENGE 07 HARD

Single Byte XOR

Break a single-byte XOR cipher using brute force or known plaintext attack.

Background

When data is XORed with a single byte, there are only 256 possible keys (0-255). This makes it trivially weak and susceptible to brute force attacks.

If you know part of the plaintext (like the flag format crypto{), you can recover the key instantly using a known plaintext attack.

Challenge

Data has been XORed with a single secret byte. Decrypt the ciphertext to reveal the flag.

Ciphertext (Hex)
73626960647f6b206821204f21254f7d694f7624662065622127234f726927756d

Hint
Remember the flag format! If you know the first character should be 'c', what key value would make that work?

Solution

Finding the Key

Since we know the flag starts with 'c' (ASCII 99 = 0x63) and the first ciphertext byte is 0x73:

First ciphertext byte: 0x73 (115 in decimal)
Expected plaintext:    'c'  (99 in decimal)

Key = 0x73 ^ ord('c')
Key = 115 ^ 99
Key = 16 (0x10)

Method 1: Known Plaintext Attack

Python 
cipher = bytes.fromhex('73626960647f6b206821204f21254f7d694f7624662065622127234f726927756d')

# Deduce key from known plaintext
key = cipher[0] ^ ord('c')  # 0x73 ^ 0x63 = 0x10 = 16
print(f"Key: {key}")

# Decrypt
plaintext = bytes(b ^ key for b in cipher)
print(plaintext)

Method 2: Brute Force

Python 
cipher = bytes.fromhex('73626960647f6b206821204f21254f7d694f7624662065622127234f726927756d')

for key in range(256):
    result = bytes(b ^ key for b in cipher)
    if result.startswith(b'crypto{'):
        print(f"Key found: {key}")
        print(result)
        break
