XOR Basics

Master the fundamental bitwise XOR operation - the building block of cryptography.

Background

XOR (Exclusive OR) is a bitwise operator that returns 1 if the bits are different, and 0 if they're the same. In Python, the XOR operator is the caret (^).

Remember: XOR returns 1 when bits are different

Challenge

Given the string label, XOR each character with the integer 13. Convert the results back to a string and submit as crypto{new_string}.

üí°
Hint
Use ord(c) to get the integer value, XOR with 13, then use chr() to convert back.

Solution

Python Code

Python 
string = 'label'
result = ''.join(chr(ord(c) ^ 13) for c in string)
print(result)  # Output: aloha
